# Resume RAG Chatbot

A Resume Question-Answering chatbot built using:

- Streamlit
- LangChain
- Gemini
- FAISS
- RAG (Retrieval Augmented Generation)

## Features

- Upload Resume PDF
- Ask Questions About Resume
- AI Powered Answers
- Vector Search using FAISS
- Chat Interface

## Tech Stack

- Python
- Streamlit
- LangChain
- Gemini API
- FAISS

## Run Locally

```bash
pip install -r requirements.txt
python -m streamlit run app.py
